- `251222_H3 8호기 발사 경과.pdf`
- `NASA awards Momentus contract for solar sail demonstration study(영).pdf`
- `위성영상가격.png`
- `인공위성_질문응답.xlsx`
- `해외정부 우주항공 현황.png`
